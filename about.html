III. Chạy chương trình

  1. View -> Server Exploder -> nhấp phải chuột vào Data Connectitons -> Add Connection... -> Microsoft SQL Server Database File -> chọn OK
  2. Sau khi hộp thoại Add Connection xuất hiện, ở mục Data file name (new or existing), chọn Browse... -> Chọn đến thư mục Source code(DoAn) -> bin -> Debug -> chọn DoAn.mdf -> chọn Test Connection ở góc trái bên dưới, khi nào chương trình báo successed, chọn Ok để đóng hộp thoại Add Connection.
  3. Sau khi Test Connection thành công, chạy chương trình như bình thường.
